## 🚀 关于项目

我们厌倦了付费订阅和不方便的宏。**Rust Tuner** 是一个开源项目，旨在通过便捷的界面让每个玩家都能使用最佳的按键绑定、FPS 优化、抄家计算和电路图。

**重要提示：** 该程序不会干扰游戏内存，也不使用按键注入 (Key Injection)。我们仅与允许的配置文件 (`client.cfg`, `keys.cfg`) 交互，这使得该程序对 **Easy Anti-Cheat (EAC) 100% 安全。**

## ✨ 主要功能

* ⌨️ **Interactive Bind Engine:** 用于配置快捷键的可视化键盘和鼠标。从自动奔跑到复杂的管理员命令。
* ⚡ **Performance Hub:** Steam 启动参数生成器、图形预设 (Potato, PVP Tryhard) 和隐藏的引擎变量管理 (`gc.buffer`, `physics.steps`)。
* 💣 **Advanced Raid Calculator:** 精确的爆炸物计算，包括与工作台绑定的 Eco 抄家（近战武器，火箭）。
* 🔌 **Interactive Electricity:** 现成的电路图、用于学习的废料计算器和制作的资源估算。
* 🌍 **Server Manager:** 保存最喜欢的服务器，记录门密码 (Code Locks) 和坐标。
