## 🚀 Sobre el proyecto

Estamos cansados de suscripciones pagas y macros incómodos. **Rust Tuner** es un proyecto de código abierto creado para dar a cada jugador acceso a los mejores binds, optimización de FPS, cálculos de raids y esquemas eléctricos en una interfaz conveniente.

**Importante:** El programa NO interfiere con la memoria del juego y NO utiliza inyección de teclas (Key Injection). Solo interactuamos con los archivos de configuración permitidos (`client.cfg`, `keys.cfg`), lo que hace que el programa sea **100% seguro para Easy Anti-Cheat (EAC).**

## ✨ Características principales

* ⌨️ **Interactive Bind Engine:** Teclado y ratón visuales para configurar atajos de teclado. Desde auto-run hasta comandos de administrador complejos.
* ⚡ **Performance Hub:** Generador de parámetros de lanzamiento de Steam, presets de gráficos (Potato, PVP Tryhard) y gestión de variables ocultas del motor (`gc.buffer`, `physics.steps`).
* 💣 **Advanced Raid Calculator:** Cálculo preciso de explosivos, incluyendo Eco-raids (armas cuerpo a cuerpo, flechas de fuego) vinculados a los bancos de trabajo.
* 🔌 **Interactive Electricity:** Esquemas listos para usar, calculadora de chatarra para aprender y estimaciones de recursos para el crafteo.
* 🌍 **Server Manager:** Guarda servidores favoritos, registra contraseñas de puertas (Code Locks) y coordenadas.
